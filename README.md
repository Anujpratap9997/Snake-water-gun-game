# Snake Water Gun Game

## Overview
The Snake Water Gun Game is a simple and engaging project designed for beginners who want to learn the fundamentals of Python programming. This game is a digital version of the classic rock-paper-scissors game with a fun twist, where the player controls a snake and has to choose between water and gun to defeat the opponent’s character.

## Objectives
- To understand the basic concepts of Python such as variables, loops, conditionals, and functions.
- To implement game logic in Python, using user input and random outcomes.
- To practice problem-solving skills and enhance programming ability through project-based learning.

## How to Play
1. The player will make a choice between 'snake', 'water', or 'gun'.
2. The opponent’s choice will be made randomly by the program.
3. The outcome will be determined based on the rules:
   - Snake beats Gun
   - Gun beats Water
   - Water beats Snake
4. The game will display the result of the round and ask if the player wants to play again.

## Getting Started
To run the game:
1. Make sure you have Python installed on your machine.
2. Clone the repository:
   ```
   git clone https://github.com/Anujpratap9997/Snake-water-gun-game.git
   ```
3. Navigate into the project directory:
   ```
   cd Snake-water-gun-game
   ```
4. Run the game:
   ```
   python snake_water_gun.py
   ```
